Box Model

Every HTML element can be considered as a box with four boxes: margin, border, padding, and content.
Heigh // height : 200px;
width // width : 300px;
Content // The content of the box, where text and images appearbr
Border // Border around the content
border-width // border-width : 10px;
border-style // border-style : solid;
border-color // border-color : red;
border:2px solid blue; // shorthand for border-width, border-style, and border-color

Border Sides

border-top // border-top : 5px solid blue;
border-right // border-right : 5px solid blue;
border-bottom // border-bottom : 5px solid blue;
border-left // border-left : 5px solid blue;
Border Radius

border-radius // border-radius : 15px;
border-top-left-radius // border-top-left-radius : 10px;
border-top-right-radius // border-top-right-radius : 10px;
border-bottom-right-radius // border-bottom-right-radius : 10px;
border-bottom-left-radius // border-bottom-left-radius : 10px;
Padding // space between the content and the border
padding-top // padding-top : 20px;
padding-right // padding-right : 20px;
padding-bottom // padding-bottom : 20px;
padding-left // padding-left : 20px;
padding: 20px; // shorthand for padding-top, padding-right, padding-bottom, and padding-left
padding : 10px 15px; // top and bottom 10px, left and right 15px
padding : 10px 15px 20px; // top 10px, left and right 15px, bottom 20px
padding : 10px 15px 20px 25px; // top 10px, right 15px, bottom 20px, left 25px
Margin // space outside the border
margin-top // margin-top : 30px;
margin-right // margin-right : 30px;
margin-bottom // margin-bottom : 30px;
margin-left // margin-left : 30px;
margin: 30px; // shorthand for margin-top, margin-right, margin-bottom, and margin-left
margin : 10px 15px; // top and bottom 10px, left and right 15px
margin : 10px 15px 20px; // top 10px, left and right 15px, bottom 20px
margin : 10px 15px 20px 25px; // top 10px, right 15px, bottom 20px, left 25px
