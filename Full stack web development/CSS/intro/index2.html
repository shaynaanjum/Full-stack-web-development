Alpha Channel

The alpha channel defines the opacity of a color. It is a number between 0.0 (fully transparent) and 1.0 (fully opaque).
Example:
rgba(255, 0, 0, 0.5); // red with 50% opacity
rgba(0, 255, 0, 0.8); // green with 80% opacity
rgba(0, 0, 255, 0.3); // blue with 30% opacity

Opacity

The opacity property sets the opacity level for an element. It is a number between 0.0 (fully transparent) and 1.0 (fully opaque).
Example:
opacity: 0.5; // 50% opacity
opacity: 0.8; // 80% opacity
opacity: 0.3; // 30% opacity

CSS Transition

The transition property is used to create smooth animations when changing CSS properties. It allows you to specify the duration, timing function, and delay of the transition.
Example:
transition: all 0.3s ease-in-out; // applies to all properties, duration 0.3 seconds, ease-in-out timing function
transition: background-color 0.5s linear; // applies to background-color, duration 0.5 seconds, linear timing function
transition: transform 0.2s ease; // applies to transform, duration 0.2 seconds, ease timing function
transition shorthand - property | duration | timing-function | delay

CSS Transform

The transform property is used to apply 2D or 3D transformations to an element. It allows you to rotate, scale, skew, or translate an element.
Example:
transform: rotate(45deg); // rotates the element 45 degrees
transform: scale(1.5); // scales the element to 1.5 times its original size
transform: skew(20deg, 10deg); // skews the element by 20 degrees on the X-axis and 10 degrees on the Y-axis
transform: translate(50px, 100px); // moves the element 50 pixels to the right and 100 pixels down
traslateX(50px); // moves the element 50 pixels to the right
translateY(100px); // moves the element 100 pixels down

Box Shadow

The box-shadow property is used to add shadow effects around an element's frame. You can set multiple shadows separated by commas.
Example:
box-shadow: 10px 10px 5px grey; // horizontal offset, vertical offset, blur radius, shadow color
box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.5); // using rgba for shadow color with opacity
box-shadow: inset 5px 5px 10px black; // inset shadow

Background Image

The background-image property is used to set an image as the background of an element. You can use various functions to control the display of the background image.
Example:
background-image: url('image.jpg'); // sets the background image
background-size: contain/cover/auto; // controls the size of the background image
background-repeat: no-repeat; // prevents the background image from repeating
background-position: center; // centers the background image

Position

The position property is used to specify the positioning method for an element. It can take the following values: static, relative, absolute, fixed, and sticky.
Example:
position: static; // default position
position: relative; // positioned relative to its normal position
position: absolute; // positioned relative to the nearest positioned ancestor
position: fixed; // positioned relative to the viewport
position: sticky; // toggles between relative and fixed, depending on the scroll position

Flexbox

Flexbox is a layout model that allows you to design complex layouts easily and efficiently. It provides a more flexible way to arrange items within a container.
Example:
display: flex; // defines a flex container
flex-direction: row/column; // defines the direction of the flex items
justify-content: center/space-between/space-around; // aligns flex items along the main axis
align-items: center/stretch/flex-start/flex-end/baseline; // aligns flex items along the cross axis
flex-wrap: nowrap/wrap/reverse; // controls whether flex items wrap onto multiple lines
align-content:flex-start/flex-end/center/space-between/space-around/evenly/baseline: //it sets the distribution of space between and around content items along a flexbox's cross-axis
align-self:flex-start/flex-end/center/baseline; //Aligns an items along the cross axis (for individual)

Flex Sizing

flex-basis:100px //It sets the initial main size of a flex item.
flex-grow:1 //It specifies how much of the flex continer's remaining space should be assigned to the flex item's main size.
flex-shrink:1 //It sets the flex shrink factor of a flex item.
flex Shorthand - flex:2 2 100px; // flex-grow|flex-shrink|flex-basis
flex:2 100px; // flex-grow|flex-basis
flex:2; //flex-grow(unitiess)
flex: 100px; //flex-basis

CSS Grid

CSS Grid is a powerful layout system that allows you to create complex grid-based layouts easily. It provides a two-dimensional grid system for arranging items in rows and columns.
Example:
display: grid; // defines a grid container
grid-template-columns: 100px 200px; // defines the columns of the grid
grid-template-rows: 50px 100px; // defines the rows of the grid
grid-gap: 10px; // sets the gap between grid items
grid-column: 1 / 3; // spans an item across columns 1 to 3
grid-row: 1 / 2; // spans an item across rows 1 to 2

Grid-Columns

grid-template-columns: repeat(3, 1fr); // creates 3 equal columns
grid-template-columns: 100px auto 200px; // creates columns with specified widths
grid-template-columns: minmax(100px, 1fr); // creates a column that is at least 100px wide and can grow to fill available space
grid-column-start: 1; // starts an item at column 1
grid-column-end: 3; // ends an item at column 3
grid-column: 1 / 3; // shorthand for grid-column-start and grid-column-end

Grid-Rows

grid-template-rows: repeat(2, 1fr); // creates 2 equal rows
grid-template-rows: 50px auto; // creates rows with specified heights
grid-template-rows: minmax(50px, 1fr); // creates a row that is at least 50px high and can grow to fill available space
grid-row-start: 1; // starts an item at row 1
grid-row-end: 2; // ends an item at row 2
grid-row: 1 / 2; // shorthand for grid-row-start and grid-row

Grid-Areas

grid-template-areas: "header header header" "sidebar content content" "footer footer footer"; // defines named grid areas
grid-area: header; // assigns an item to a named grid area
grid-area: sidebar; // assigns an item to a named grid area
grid-area: content; // assigns an item to a named grid area
grid-area: footer; // assigns an item to a named grid area

Common Properties

Horizontal

justify-items // aligns items horizontally in a grid container
justify-content // aligns the entire grid horizontally within the container
justify-self // aligns an individual item horizontally within its grid area

Vertical

align-items // aligns items vertically in a grid container
align-content // aligns the entire grid vertically within the container
align-self // aligns an individual item vertically within its grid area

Horizontal + Vartical

place-items // shorthand for justify-items and align-items
place-content // shorthand for justify-content and align-content
place-self // shorthand for justify-self and align-self
