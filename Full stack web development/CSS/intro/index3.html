CSS Animations

  • Animation-name

    • The animation-name property specifies the name of the @keyframes animation to apply to an element. It is used in conjunction with other animation properties to create animations.
    Example:
    animation-name: example; // applies the 'example' animation defined in @keyframes
    animation: example 5s infinite; // shorthand for applying the 'example' animation with a duration of 5 seconds and infinite iterations

  • Animation-duration

    • The animation-duration property specifies the length of time an animation takes to complete one cycle. It is defined in seconds (s) or milliseconds (ms).
    Example:
    animation-duration: 2s; // sets the duration of the animation to 2 seconds
    animation-duration: 500ms; // sets the duration of the animation to 500 milliseconds

  • Animation-timing-function

    • The animation-timing-function property specifies the speed curve of an animation. It defines how the intermediate values of the animation are calculated over time.
    Example:
    animation-timing-function: linear; // animation progresses at a constant speed
    animation-timing-function: ease; // animation starts slow, speeds up, and then slows down at the end
    animation-timing-function: ease-in; // animation starts slow and speeds up towards the end
    animation-timing-function: ease-out; // animation starts fast and slows down towards the end
    animation-timing-function: cubic-bezier(0.1, 0.7, 1.0, 0.1); // custom timing function using cubic-bezier curve

  • Animation-delay

    • The animation-delay property specifies the amount of time to wait before starting an animation. It is defined in seconds (s) or milliseconds (ms).
    Example:
    animation-delay: 1s; // delays the start of the animation by 1 second
    animation-delay: 500ms; // delays the start of the animation by 500 milliseconds

  • Animation-iteration-count

    • The animation-iteration-count property specifies the number of times an animation should be played. It can be a specific number or 'infinite' for continuous looping.
    Example:
    animation-iteration-count: 3; // plays the animation 3 times
    animation-iteration-count: infinite; // plays the animation indefinitely

  • Animation-direction

    • The animation-direction property specifies whether an animation should play in reverse on alternate cycles. It can be set to 'normal', 'reverse', 'alternate', or 'alternate-reverse'.
    Example:
    animation-direction: normal; // plays the animation in the normal direction (default)
    animation-direction: reverse; // plays the animation in reverse direction
    animation-direction: alternate; // alternates between normal and reverse direction on each cycle
    animation-direction: alternate-reverse; // alternates between reverse and normal direction on each cycle

Keyframes

Keyframes are used to define the intermediate steps in a CSS animation. They allow you to specify the styles at various points during the animation sequence.
Example:
@keyframes example {
0% { background-color: red; }
50% { background-color: yellow; }
100% { background-color: green; }
}
@keyframes example {
from { transform: rotate(0deg); }
to { transform: rotate(360deg); }
}
animation: example 5s infinite; // applies the animation to an element, duration 5 seconds, infinite loop
animation: example 2s ease-in-out 1s 3 reverse; // applies the animation with specific timing, delay, iteration count, and direction

Shorthand

The animation property is a shorthand for setting all the animation properties at once. It can include the animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, and animation-direction.
Example:
animation: example 5s ease-in-out 1s 3 reverse; // applies the 'example' animation with a duration of 5 seconds, ease-in-out timing function, 1 second delay, 3 iterations, and reverse direction
animation: example 2s linear infinite; // applies the 'example' animation with a duration of 2 seconds, linear timing function, and infinite iterations

Media Queries

Media queries are used to apply different styles to different devices or screen sizes. They allow you to create responsive designs that adapt to various screen widths, orientations, and resolutions.
Example:
@media (max-width: 600px) {
body { background-color: lightblue; }
}
@media (min-width: 601px) and (max-width: 1200px) {
body { background-color: lightgreen; }
}
@media (min-width: 1201px) {
body { background-color: lightcoral; }
}

Z-index

The z-index property specifies the stack order of elements. It determines which elements are displayed in front of or behind other elements. Elements with a higher z-index value will be displayed in front of those with a lower value.
Example:
.element1 { z-index: 1; } // element1 will be behind element2
.element2 { z-index: 2; } // element2 will be in front of element1
.element3 { z-index: 0; } // element3 will be behind both element1 and element2
z-index:auto(0) // default value, elements will be stacked in the order they appear in the HTML
z-index:1/2/... // positive values will stack in front of elements with lower z-index values
z-index:-1/-2/... // negative values will stack behind elements with higher z-index
